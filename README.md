# `heatmap.sh` - Heatmap cli

[heatmaps in your terminal](http://en.wikipedia.org/wiki/Heat_map)

```
X     Z
|   /
|  /
| /
|/
*____ Y
```

Use:

*  ░ Light shaded block
*  ▒ Medium shaded block
*  ▓ Dark shaded block
*  █ Darker shaded block

Random sample output:

```
░▒▓░█▒
▒▓▓█▓▓
█▓▓▓▒▓
▒▓▒█▓▒
░▒▓███
```
